# MACD股票筛选回测系统

## 项目简介

这是一个基于MACD指标的全市场股票月度/年度回测系统。系统从PostgreSQL数据库获取股票数据，计算MACD指标，按月运行完整交易循环（买入→持有→卖出→立即重新选股），统计每月的收益率、胜率等指标。

## 功能特点

- 从PostgreSQL数据库读取全市场股票数据（SQLAlchemy连接池）
- 自动计算MACD指标（DIFF、DEA、MACD柱）
- 支持talib库加速计算（如未安装则自动使用手动计算）
- **完整交易循环**：金叉买入 → MACD下降卖出 → 卖出后立即重新选股买入
- **多重过滤条件**：主板股票、前一日MACD<0、前5日MACD不为持续很小、非涨停、非ST
- **历史模拟选股**：在回测当日先对候选股票做历史模拟回测，选取历史收益最高的股票买入
- **逐月回测**：每月独立初始资金10000元，逐月统计输出 + 年度汇总
- 手续费：万三双向 + 印花税千一（卖出时）
- 支持批量查询和进度显示

### 3. 数据库表结构

系统使用以下两个表：

**日线数据表（默认：`trade_history_daily`）**

```sql
CREATE TABLE trade_history_daily (
    ts_code VARCHAR(10),      -- 股票代码（如 000001.SZ）
    trade_date DATE,           -- 交易日期
    open DECIMAL(10,3),       -- 开盘价
    close DECIMAL(10,3),      -- 收盘价
    high DECIMAL(10,3),        -- 最高价
    low DECIMAL(10,3),         -- 最低价
    vol BIGINT,               -- 成交量
    PRIMARY KEY (ts_code, trade_date)
);
```

**股票基本信息表（默认：`stock_basic`）**

```sql
CREATE TABLE stock_basic (
    ts_code VARCHAR(10),      -- 股票代码
    name VARCHAR(50),          -- 股票名称（用于判断ST）
    ...
);
```

如果表名不同，请在 `config.py` 中修改 `TABLE_NAME`。

## 策略逻辑

### 选股条件

1. **MACD基础条件**：DIFF > 0, DEA > 0, 且出现金叉（DIFF上穿DEA）
2. **主板过滤**：仅交易沪市600/601/603/605开头和深市000/001开头股票（排除科创板、创业板）
3. **前一日MACD为负**：买入当天前一日MACD < 0
4. **前5日MACD不为持续很小**：前5个交易日MACD不能一直接近零
5. **非涨停**：前一日涨幅不超过9.9%
6. **非ST**：排除ST/*ST股票

### 交易规则

1. **买入**：满足选股条件的当天，以**开盘价**买入
2. **持有**：持续持有，每日检查是否触发卖出条件
3. **卖出**：买入后MACD柱（MACD值）开始下降的那一天，以**收盘价**卖出
4. **循环**：卖出后立即重新选股买入，当月内循环交易直到回测结束
5. **月末强制平仓**：月度回测结束时若仍持有，以最后一天收盘价强制卖出

### 历史模拟选股

在每个选股日，发现金叉股票后，系统会做一次历史模拟：
- 在回看过去30天窗口内，模拟最近一次金叉买入、MACD下降卖出的收益
- 从候选股票中选取历史模拟收益最高的股票实际买入
- 如果模拟收益最高的股票不满足过滤条件（如前一日MACD>=0、涨停、ST等），则跳过并尝试次优股票

## 自定义配置

### 修改MACD参数

在 `config.py` 中修改MACD计算参数：

```python
MACD_CONFIG = {
    'fast_period': 12,     # 快速EMA周期
    'slow_period': 26,    # 慢速EMA周期
    'signal_period': 9     # Signal线周期
}
```

### 修改回测年份

在 `main.py` 中修改回测年份：

```python
year = 2025  # 改为想要回测的年份
```

### 修改初始资金

在 `monthly_backtest.py` 的 `MonthlyBacktest` 类中修改：

```python
def __init__(self, initial_capital: float = 10000):
    self.initial_capital = initial_capital
```

## 依赖说明

- `psycopg2-binary`: PostgreSQL数据库连接
- `sqlalchemy`: SQLAlchemy连接池
- `pandas`: 数据处理和分析
- `numpy`: 数值计算
- `tqdm`: 进度条显示
- `talib`（可选）：技术分析库，如安装将优先使用，否则使用手动计算

## 注意事项

1. 确保数据库中已有足够的历史数据（建议至少6个月），MACD计算需要至少34天数据
2. 实际使用前请充分回测，过去的业绩不代表未来表现
3. 历史模拟选股模块依赖30天回看窗口，需要该股票有足够的历史数据
4. 主板过滤规则可根据需要调整为包含科创板、创业板等板块
